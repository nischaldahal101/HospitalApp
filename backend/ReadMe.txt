Tech Used for Backend:
1. Mongodb
2. Express Server
3. Bcryptjs - for encrypting the password
4. JSONwebtoken - for generating token while logging in the Server
5. nodemon - to watch changes in the files and restart the server automatically
6. express-validator -  used to validate the json data/data that users inputs from frontend.
7. Deployed on HEROKU
8. GridFs System- - used to store image using gridfs mongo bucket system.
9. Multer - storage engine system to store image.
10. ***Note*** only 512GB data space in free cluster on mongoDB atlas. -- Image files takes space that txt.
